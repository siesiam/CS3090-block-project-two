# CS3090-block-project-two
<p>This program takes user input and checks its strength as a password.</p>

<p>To run the program, all the user needs to do is first download the BlockProjectTwo.java file to their computer. Once that is done, the user will upload it to a compatiable IDE, such as Eclipse, and run it as normal. Once asked by the program, the user will enter a password that they want to check the strength of. After, the program will report whether the strength of the password is really weak, weak, medium, strong, or really strong. Then, it will end the program.</p>

<p>This tool is for educational use only and should not be used for securing sensitive information. It only verifies the strength of the password based on the length and how many different type of characters it has. This only includes lowercase letters, uppercase letters, numbers, and a character that is not the same as the other types mentioned. It does not consider common words or phrases when checking the strength.</p>

<p>This program is meant to check the strength of the password only. Due to this, it does hold the exact password within the program. To address any ethical considerations, because the exact password is held within the program, this code could be modified to save any input to be seen by individuals other than the user. For responsible use, do not enter any sensitive information such as information used for personal identification, what you may use in current passwords, or anything else of similar security threat.</p>
